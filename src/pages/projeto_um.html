Projeto 1 - Página de Login

Descrição:

Projeto inicial desenvolvido durante a aula de Frontend exemplificando uma página de login. Apresenta uma imagem de fundo e uma caixa central com informações básicas de um formulário, utilizadas para o usuário realizar o login desejado. O botão é fictício e não envia o formulário realmente.

Tecnologias Utilizadas:

HTML, CSS, JavaScript, Bootstrap
